# SSHBool

Native infrastructure workspace — Tauri v2 + Rust + React (Vite).

## Develop

```bash
bun install
bun run tauri dev
```

## Quality

```bash
bun run typecheck
bun run lint
bun run test
cargo test --manifest-path src-tauri/Cargo.toml --workspace --lib
```
